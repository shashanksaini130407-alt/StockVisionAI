📈 About StockVision AI

An AI-powered stock market analysis and prediction platform designed to help investors make smarter financial decisions using Machine Learning, Technical Indicators and Interactive Analytics.

🚀 Project Overview

StockVision AI combines Artificial Intelligence, Financial Analytics and Interactive Visualization into one intelligent platform.

Users can search stocks, predict future prices, compare companies, manage portfolios, analyze market trends and receive AI-powered BUY/SELL recommendations.

⭐ Key Features

📊 Stock Prediction

Predict future stock prices using Machine Learning algorithms.

📈 Analytics Dashboard

Interactive charts, statistics, trends and performance insights.

⚖ Stock Comparison

Compare multiple companies side-by-side with AI recommendations.

💼 Portfolio Management

Track investments, profits, returns and portfolio allocation.

⭐ Watchlist

Monitor favourite stocks with AI-powered recommendations.

📜 Prediction History

Search, export and review all previous predictions.

⚙ How StockVision AI Works

  1. Select a stock symbol.
  2. Download historical stock data.
  3. Generate technical indicators.
  4. Prepare machine learning features.
  5. Train Random Forest model.
  6. Predict future stock prices.
  7. Generate BUY / SELL recommendation.
  8. Display charts and analytics.
